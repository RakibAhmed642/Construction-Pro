# **App Name**: ConstructView

## Core Features:

- Project Overview: Displays key project metrics like budget, timeline, and progress.
- Task Management: Allows users to create, assign, and track tasks.
- Document Repository: Centralized storage for project documents with version control.
- Real-time Chat: Enables instant communication between team members. Allows files upload directly from the chat window
- Progress Forecasting: Use AI tool that looks at existing progress and produces estimates to predict potential delays and resource needs. Incorporates historical project data.
- Budget Tracking: Monitors expenses against the budget and identifies potential overruns.
- Reporting: Generates customizable reports on project status, financials, and resource allocation.

## Style Guidelines:

- Primary color: Navy blue (#2E3148) to convey professionalism and stability.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Teal (#388E8E) to highlight important actions and metrics.
- Body text: 'PT Sans' for a clear and readable interface. 
- Headlines: 'Space Grotesk' (sans-serif) to contrast body and call out headlines.
- Use clean, modern icons to represent project elements.
- A card-based layout will organize the dashboard elements for easy navigation and accessibility.
- Subtle transitions and animations to enhance user experience when tasks or information are updated.
